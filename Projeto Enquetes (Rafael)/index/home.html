Bem-vindo ao Sistema de Enquetes

Este é um sistema simples para a criação, votação e gestão de enquetes. Use os links acima para navegar pelas diferentes funcionalidades do sistema.

O que você pode fazer aqui:

  • Cadastrar-se para criar enquetes ou votar em enquetes existentes.
  • Fazer login para acessar sua conta e ver enquetes que criou.
  • Criar novas enquetes para perguntar algo para os usuários.
  • Votar nas enquetes disponíveis.
  • Ver resultados de enquetes que já foram finalizadas.
  • Acessar o painel de administração para gerenciar enquetes.
